# Сравнительный анализ номинальных и реальных потребительских расходов населения России (2018–2026)

## Описание проекта

Проект посвящён анализу потребительских расходов населения России на основе данных платформы СберИндекс за период с 2018 по 2026 год.

Цель исследования — провести сравнительный анализ номинальных и реальных потребительских расходов, выявить кризисные периоды, исследовать структуру потребления и проверить ряд статистических гипотез.

Проект выполнен в рамках дисциплины **«Анализ данных на Python»**.

---

## Цель исследования

Провести сравнительный анализ номинальных и реальных потребительских расходов населения России и выявить основные закономерности их изменения во времени.

---

## Источник данных

Для исследования использовались данные платформы **СберИндекс**, полученные программным способом через API.

Использовались следующие наборы данных:

* `consumer-spending` — номинальные потребительские расходы;
* `consumper-spending-index-sa` — индекс реальных потребительских расходов.

Период исследования: **2018–2026 годы**.

## Этапы выполнения работы

### 1. Сбор данных

Данные были получены через API платформы СберИндекс и объединены в единый набор данных.

### 2. Предобработка данных

На этапе подготовки были выполнены:

* преобразование типов данных;
* проверка пропусков;
* проверка дубликатов;
* объединение таблиц;
* формирование итогового датасета.

### 3. Создание новых признаков

Для дальнейшего анализа были рассчитаны:

* год наблюдения;
* месяц наблюдения;
* квартал;
* темпы роста номинальных расходов;
* темпы роста реальных расходов;
* показатель gap (разница между темпами роста номинальных и реальных расходов).

### 4. Анализ данных

В рамках исследования были выполнены:

* выявление кризисных месяцев;
* корреляционный анализ;
* анализ структуры потребительских расходов;
* анализ волатильности категорий расходов;
* кластеризация месяцев по темпам роста;
* проверка статистических гипотез.

---

## Проверяемые гипотезы

### Гипотеза 1

Сезонная структура общественного питания совпадает с сезонностью продовольственных товаров.

### Гипотеза 2

Реальные расходы восстанавливаются после кризисов с той же скоростью, что и номинальные расходы.

### Гипотеза 3

Выбросы показателя gap предшествуют выбросам индекса реальных расходов.

---

## Основные результаты

В ходе исследования:

* были выявлены кризисные месяцы на основе статистического подхода;
* исследована связь между номинальными и реальными расходами;
* определены наиболее и наименее волатильные категории расходов;
* выявлены категории, наиболее чувствительные к кризисным периодам;
* проведена кластеризация месяцев по характеру изменения расходов;
* проверены исследовательские гипотезы.
